Image without width limit

An example of using <img-fixheight>, the image content is cropped and centered, with a fixed height of 560px at 1x. Width corresponds to page width. You can use your favorite developer tools to use this page at different widths, scale factors, and see how the banner is responsive.

Image with width limit

Same example than previous, but with limitmaxwidth attribute. This will use the maximum value specified in widths and limit the maximum width to it, whatever scale factor is used.
